# Càlcul de Mates per a estudiants de primària.

Programa per a pràcticar càlcul básic.
Escolliu entre sumes, restes, multiplicacions i divisions o feu una serie de 20 operacions i repetiu les que no hagueu encertat.

Creat a l'estiu de 2025 com a pràctica de Python amb suport de Copilot. Pujat i compartit a Streamlit el juliol de 2026 amb suport de Claude
